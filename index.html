Défi GitHub : Automatiser le tri de mes fichiers de cours en Python

Avec le rythme de la prépa, mon dossier "Téléchargements" est rapidement devenu un chaos sans nom entre les PDF de maths, les énoncés de physique et les scripts de TP. Au lieu de tout ranger à la main, j'ai décidé de créer un script d'automatisation en Python.

Le principe du script est de surveiller un dossier spécifique, d'analyser l'extension de chaque nouveau fichier (comme .pdf, .py, .docx) et de les déplacer instantanément dans des sous-dossiers correspondants et datés.

Ce que ce défi m'a appris

J'ai dû explorer la bibliothèque système os et le module shutil de Python. Gérer les cas particuliers, comme deux fichiers qui portent exactement le même nom sans écraser le premier, était un super exercice de logique pure.

02 Juin 2026

Résolution de problème : Optimiser un algorithme de recherche

Cette semaine, je me suis attaquée à un défi d'algorithmie classique sur GitHub qui consistait à trouver le chemin le plus court dans un labyrinthe représenté sous forme de matrice (un graphe).

Ma première tentative utilisait une approche naïve en force brute, mais la complexité algorithmique faisait ramer le programme dès que la taille de la matrice dépassait 50x50.

Le passage à l'optimisation

Pour résoudre ce problème d'efficacité, j'ai implémenté un algorithme de parcours en largeur (BFS) en utilisant une structure de file (queue). Cela a permis de réduire drastiquement le nombre de chemins explorés inutilement.

C'est ce genre de challenge d'optimisation de code et de gestion de la mémoire qui me passionne le plus dans mes cours d'informatique.

22 Mai 2026

Dompter le langage C : Ma bataille avec les pointeurs et la mémoire

Pour notre projet d'architecture système, on a dû abandonner le confort de Python pour coder en langage C. L'objectif : allouer manuellement de la mémoire pour gérer une liste chaînée de données.

Au début, mon terminal n'affichait qu'une seule chose en boucle : la fameuse erreur Segmentation Fault, signe qu'un de mes pointeurs essayait d'accéder à une zone mémoire interdite.

Comprendre ce qu'il se passe sous le capot

En utilisant un outil de débogage nommé Valgrind, j'ai enfin pu tracer mes allocations de mémoire (malloc) et m'assurer que chaque octet ouvert était bien refermé avec un free. C'est difficile, mais super formateur pour comprendre comment un ordinateur fonctionne réellement.

Un script pour analyser mes propres notes de prépa avec Pandas

Pour lier l'utile à l'agréable, j'ai exporté toutes mes notes de l'année au format CSV et j'ai écrit un script en Python utilisant la bibliothèque de Data Science Pandas pour en faire une analyse statistique complète.

Mon script calcule mes moyennes pondérées par coefficient, extrait mes écarts-types par matière, et génère des prévisions sur les notes minimales à obtenir aux prochains partiels pour valider mon semestre avec mention.

Défi Cryptographie : Décoder un message chiffré (Algo de César avancé)

J'ai participé à un mini-défi de sécurité informatique lancé sur un dépôt GitHub étudiant. Le but était de casser un message secret sans avoir la clé de chiffrement, en utilisant la méthode de l'analyse de fréquence des lettres.

Comme la lettre "E" est la plus fréquente en français, j'ai codé un programme qui compte les occurrences de chaque caractère dans le texte crypté pour deviner automatiquement le décalage de l'alphabet utilisé.
